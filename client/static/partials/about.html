Summary of Each Partial / Final Thoughts

home
  • Static contents with the custom stylings with the help of Bootsrap's jumbotron class

me
  • Static contents with the custom stylings
  • CERCA TROVA hover event is handled by the combination of the following AngularJS directives: ng-mouseover / ng-mouseleave / ng-show / ng-hide / ng-style
resume
  • Custom styling on the anchor to look like buttons
  • Styling the fonts on a button hover is triggered by ng-mouseleave / ng-mouseover / ng-style of AngularJS directives
contact
  • Custom styling on the buttons
  • ng-click / ng-hide / ng-show directives to change the buttons to reveal the contact information

Final Thoughts
  • Total duration to complete this website from the basic sketch to the final launch: 8 days
  • What I like about this website
    • MVC architecture to develop organized codes / neat folder structures
    • Easy and great model structure, organized and clean codes, and no unnecessary methods or files
    • Fully responsive web design to accommodate the smaller screens and devices
    • Additional time and care to create the custom styling and design to develop more unique contents
    • My first official project that is not an assignment from the coding bootcamp
  • What I could have done better
    • I should have used CSS preprocessor such as Sass or LESS to take its advantages of the nesting and variables which could have reduced the codes in style.css
    • Navigation bar is in each partial html. I should have put it in the main view html and use a main controller to activate each partial and styling
    • Purpose of the login for the users. Registration and login are to show that I can create backend validations and models. Other than that, it has no purpose for the users
  • I can't wait to make more projects!
  • I also can't wait to work and to evolve more as a professional software developer
